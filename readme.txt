Проект выполнил кандидат на стажировку
Мамлеев Э.С.

Данное приложение принимает 3 числа:
Первое число для количества потоков генераторов событий;
Второе число для количества событий на каждом из потоков;
Третье число для количества обработчиков событий.

После чего возвращает общее количество: 
чисел со всех потоков;
простых чисел среди поступивших со всех потоков.

Для запуска данного приложения можно открыть файл multithreadapp.exe обычным способом как исполняемый файл, 
либо открыть файл Source.cpp и скомпилировать его, что так же должно привести к запуску программы.
